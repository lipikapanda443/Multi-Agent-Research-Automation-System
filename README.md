# Multi-Agent Research Automation System

## Overview
An AI-powered research automation system that simplifies academic literature reviews using Google Gemini and the arXiv API.

## Features
- Topic Refinement
- Research Paper Retrieval
- AI-powered Paper Summarization
- Research Gap Analysis
- PDF Report Generation

## Technologies Used
- Python
- Google Colab
- Google Gemini API
- arXiv API
- FPDF

## How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Add your Gemini API key.
4. Run all cells.
5. Enter a research topic.
6. Download the generated PDF report.
